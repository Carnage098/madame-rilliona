MADAME RILLIONA V3.1 — INSTALLATION PROPRE

1. Remplace entièrement l'ancienne version par le contenu de cette archive.
2. Ne conserve pas les anciens fichiers ou dossiers d'une autre version.
3. Railway : Root Directory vide, Start Command = python bot.py
4. Variables de découverte :
   RANDOM_DISCOVERY_ENABLED=true
   RANDOM_DISCOVERY_INTERVAL_SECONDS=60
   RANDOM_DISCOVERY_MAX_ATTEMPTS=8
   RANDOM_DISCOVERY_INITIAL_DELAY_SECONDS=30
5. Supprime l'ancienne variable RANDOM_DISCOVERY_INTERVAL_MINUTES si elle existe.
6. Redéploie le dernier commit.

La découverte vérifie l'identifiant officiel dans PostgreSQL avant insertion.
Une carte déjà présente est ignorée et un autre tirage est tenté.

MADAME RILLIONA V2.9 — INSTALLATION PROPRE

1. Remplacer entièrement l'ancienne version par le contenu de cette archive.
2. Conserver à la racine : bot.py, config.py, database_manager.py et les dossiers cogs, models, repositories, services, utils.
3. Railway : Start Command = python bot.py ; Root Directory = vide.
4. Variables minimales : DISCORD_TOKEN, DATABASE_URL, GUILD_ID.
5. Variables staff conseillées :
   STAFF_ROLE_IDS=ID_ROLE_STAFF,ID_AUTRE_ROLE
   MAX_STAFF_IMAGE_BYTES=10485760
6. Conserver CARD_IMAGE_DIRECTORY=/app/data/card_images.
7. Monter un volume persistant sur /app/data/card_images pour conserver les PNG.
8. Redéployer le dernier commit.
9. Tester /base ajouter_carte puis /base verifier_carte.

Le schéma PostgreSQL est migré automatiquement au démarrage.
